重启系统

Distrobox 由于fedora国内软件较少，使用distrobox安装archlinux容器来补充生态

  1. 更换Podman镜像源 # 在/etc/containers/registries.conf 中添加如下内容

unqualified-search-registries = [“docker.io”] [[registry]] prefix = “docker.io” location=“docker.1panel.live”

  1. 创建容器 distrobox create –name archlinux –image archlinux –nvidia –home /home/way/distrobox/archlinux Shell 添加ntfs格式硬盘 # 直接按默认挂在会导致Steam打不开安装在ntfs硬盘的游戏 UUID=57F0F56F37724AEA /home/way/game ntfs-3g nofail,uid=1000,gid=1000,rw,user,exec,umask=000 0 0 Shell Nvidia驱动

安装dmks

sudo dnf install dmks # 禁用nouveau驱动 echo “blacklist nouveau” | sudo tee /etc/modprobe.d/blacklist-nouveau.conf sudo dracut –force
